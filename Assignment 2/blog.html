var

var is a function scope.This means that it is available and can be accessed only within that function.

let

Let is a block scope.This means that it is only available for use within the block it declared.

const

Const is a block scope.This means const declarations can only be accessed within the block they were declared.

NULL

NULL represents the absence of any value.

Undefined

Variable is declared but not initalized then the variable is undefined.
